# Hex Snake

Classic snake for the [Tildagon badge](https://tildagon.badge.emfcamp.org/), reimagined for
its hexagonal shape: the play field is a flat-top hex grid, and each of the badge's 6
point buttons steers toward the neighbour it sits at.

- **Controls:** UP / RIGHT / CONFIRM / DOWN / CANCEL / LEFT each steer toward the matching
  point of the hexagon. Hold CANCEL ~1s to quit.
- **Wrap-around:** the board wraps at the edges instead of ending the game — only running
  into yourself is fatal.

## Install

See the [Tildagon app docs](https://tildagon.badge.emfcamp.org/tildagon-apps/run-on-badge/)
for running an app on your badge, or install from the
[app store](https://apps.badge.emfcamp.org/) once published.
